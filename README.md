
# Getting Started with TA FASTAPI

## Prequerities
- Python 3.8

## To Start (First Time)
First create VENV
```
python -m venv ta-theo-env
.\ta-theo-env\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```

Then run the app
```
py main.py
```

## To Start (After Instalation)
```
.\ta-theo-env\Scripts\activate
py main.py
```

## To Update Model
Paste the models version
